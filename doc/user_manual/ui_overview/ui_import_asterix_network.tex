\subsection{Import ASTERIX from Network}
\label{sec:ui_import_asterix_network}


This task allows importing of ASTERIX data from the network into the opened database, and switches from Offline to Live mode. \\

\begin{figure}[H]
  \center
    \hspace*{-0.5cm}
    \includegraphics[width=17cm]{figures/asterix_import_data_network.png}
  \caption{Import ASTERIX Data from network}
\end{figure}

Except for the ASTERIX source at the top, everything is the same as described in \nameref{sec:ui_import_asterix}. \\

When the import is started, data is continously read from the network (according to the data source network lines specficiation) and processed according to \nameref{sec:live_mode}.
